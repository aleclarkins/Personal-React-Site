## 🛠 Installation & Set Up

1. Navigate to project directory and install all modules and dependencies

   ```sh
   npm install
   ```

2. Run app in developer mode

   ```sh
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
